# affiliate-ai (V2)

> AI-driven affiliate marketing engine for **Shopee** (Thailand). Closed-loop self-learning:
> scrape → translate → score → generate variants → publish → track clicks → learn → tune.

**Status:** Sprint 30 complete · Production-live at https://price-th.com

---

## What this system does, end-to-end

1. **Scrape** Shopee for trending products 4× per day (08/13/19/22 BKK)
2. **Translate** every product to EN/ZH/JA via Claude Haiku (~$0.0016/product)
3. **Render** a static multilingual site (200 products × 4 langs = 850 pages, 8 categories + client-side search)
4. **Auto-deploy** to Cloudflare Pages within 5 min of each scrape
5. **Detect promos** (price drops, discount jumps, sold surges) every 30 min
6. **Generate marketing copy** (3 angles × 2 channels) for promo products
7. **Quality gate** every variant (toxicity / disclosure / forbidden words)
8. **Auto-publish** to FB / IG / TikTok every 30 min during 8 AM–10 PM BKK *(activates when META/TikTok tokens land)*
9. **Track clicks** through Pages Function → Cloudflare Tunnel → DB → 302 to Shopee
10. **Learn nightly**: deactivate underperforming variants, rebalance scrape budget per niche

Closed loop, no human in the chain after activation.

## Live infrastructure

| Where | What |
|---|---|
| **Droplet (DigitalOcean)** | Postgres 16 + 3 systemd services (scheduler, redirect-server, cloudflared) |
| **Cloudflare Pages** project `shopee-aggregator` | 850-page static site + 1 Pages Function for `/go/<id>` |
| **Cloudflare Tunnel** `affiliate-ai-redirect` | api.price-th.com → localhost:3001 (encrypted, no public port) |
| **Domain** | price-th.com (root) + api.price-th.com (tunnel) |

11 cron jobs run 24/7 — see [docs/HANDOFF.md](docs/HANDOFF.md) §3.

## Tech stack

| Layer | Choice | Why |
|---|---|---|
| Runtime | **Bun** 1.3+ | Native TS, fast cold start, single binary |
| Language | **TypeScript** strict | One language across server + scraper + site builder |
| DB | **Postgres 16** self-host | Localhost = zero latency, $0 marginal cost |
| ORM | **Drizzle** | Type-safe, no codegen, easy migrations |
| LLM | **Anthropic Claude** Haiku 4.5 (bulk) / Sonnet 4.6 (decisions) | Best Thai language, prompt-cache friendly |
| Scraper | **Apify** `xtracto/shopee-scraper` | The only path through Shopee bot defenses |
| Web | **Static HTML** generated by Bun (V1 Tailwind theme ported) | No framework runtime, sub-second rebuilds |
| Edge | **Cloudflare Pages + Functions + Tunnel** | Free CDN + Workers + secure tunnel, all under one CF account |
| Cron | **croner** in-process | One systemd service runs everything |

## Quick start (operator)

After cloning to a fresh droplet:

```bash
# 1. Install Bun
curl -fsSL https://bun.sh/install | bash && exec $SHELL

# 2. Install deps
cd /root/affiliate-ai && bun install

# 3. Database
sudo bash scripts/setup-postgres.sh   # creates botuser + db
bun run db:push                       # apply schema

# 4. Configure .env
cp .env.example .env
chmod 600 .env
# (see docs/env-setup.md for what each var does)

# 5. Install systemd units
sudo cp deploy/systemd/*.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable --now affiliate-ai-scheduler affiliate-ai-redirect

# 6. Cloudflare Tunnel (one-shot from CF Dashboard token)
cloudflared service install <CONNECTOR_TOKEN>

# 7. First deploy
bun run deploy:site
```

Detailed onboarding: [docs/HANDOFF.md](docs/HANDOFF.md). Activation when API keys arrive: [docs/ACTIVATION.md](docs/ACTIVATION.md).
